git:

|              |                                                    |
| ------------ | -------------------------------------------------- |
| **feat**     | 新功能 (feature)                                   |
| **fix**      | 修补 bug                                           |
| **docs**     | 文档修改 (documentation)                           |
| **style**    | 格式变动 (不影响代码运行的变动，如空格、分号等)    |
| **refactor** | 重构 (既不是新增功能，也不是修改 bug 的代码变动)   |
| **perf**     | 性能优化 (performance)                             |
| **test**     | 增加测试或修改测试用例                             |
| **chore**    | 构建过程或辅助工具的变动 (如安装依赖、修改配置)    |
| **revert**   | 回退之前的 commit                                  |
| **build**    | 影响构建系统或外部依赖的更改 (如 gulp, npm, maven) |
| **ci**       | 修改 CI 配置文件和脚本                             |
